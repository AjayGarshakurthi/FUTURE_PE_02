## 📌 Project Overview
This project is a **Salon UGC (User Generated Content) Content Kit Website** designed to generate marketing content for salon businesses.

It provides ready-to-use:
- 🔥 Viral UGC hooks
- 🎬 Full video ad scripts
- 📣 High-conversion CTAs
- 📱 Instagram captions

The goal is to help salons and creators easily generate **marketing-ready content for social media campaigns**.

---

## ✨ Features
- 🎯 10 scroll-stopping UGC hooks for salon marketing  
- 🎥 3 structured UGC ad scripts (Problem → Story → Solution → Result → CTA)  
- 📣 10 high-conversion CTA lines  
- 📸 3 Instagram-ready captions  
- 📋 One-click copy functionality for content  
- ⚡ Fully interactive UI using JavaScript  
- 📱 Responsive and modern layout
